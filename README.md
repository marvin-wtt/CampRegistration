## Registration

- Reserved keys:
  - `waiting_list`
  - `leader`
  - `room_id`
  - `room_name`
- Default keys:
  - `first_name` - The first name of a participant
  - `last_name` - The last name of a participant
  - `full_name` - The full name of a participant
  - `date_of_birth` - The date of birth of a participant
  - `age` - The age of a participant
